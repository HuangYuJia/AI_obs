# OBS Virtual Try-On 使用教程

## 工作原理

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   OBS 直播画面   │ ──→ │   AI 处理引擎    │ ──→ │  换装后的画面    │
│  (人物视频流)    │     │  (虚拟试穿模型)   │     │  (输出到虚拟摄像头)│
└─────────────────┘     └─────────────────┘     └─────────────────┘
                              ↑
                        ┌─────────────────┐
                        │   服装图片       │
                        │  (拖拽上传)      │
                        └─────────────────┘
```

## 操作步骤

### 第一步：连接 OBS
1. 打开 OBS Studio
2. 启用 WebSocket：**工具 > WebSocket 服务器设置** → 勾选「启用 WebSocket 服务器」
3. 在网页顶部输入密码（如果设置了密码），点击「OBS 连接」
4. 看到「已连接」状态表示成功

### 第二步：上传服装图片
**方法一：拖拽上传**
- 从左侧图片库拖拽图片到右侧「服装图片」区域

**方法二：本地上传**
- 点击「服装图片」区域，选择本地图片

### 第三步：生成换装效果
- 点击「连接并开始」按钮
- AI 会自动将服装应用到直播画面中的人物上
- 生成结果会显示在中间预览区

## 界面说明

| 区域 | 功能 |
|------|------|
| **左侧** | 图片搜索库（用于查找服装图片） |
| **中间** | OBS 直播画面预览 + AI 生成结果 |
| **右侧** | 换装控制面板 |

### 右侧面板详解

| 项目 | 说明 |
|------|------|
| **模式选择** | 试穿 / 换装 / AI生成（三种模式） |
| **服装图片** | 要让人物穿上的衣服图片 |
| **人物图片** | 可选：上传人物照片（如不上传，使用 OBS 直播画面） |
| **提示词** | 可选：描述生成效果的关键词 |

## 常见问题

**Q: 为什么需要「人物图片」？**
A: 如果你没有开直播，可以上传一张人物照片来测试换装效果。如果已经在直播，系统会自动使用直播画面。

**Q: 为什么生成效果不好？**
A: 目前使用的是占位模型。要获得真实效果，需要接入专业的 VTON 模型（如 IDM-VTON、OOTDiffusion）。

**Q: 支持哪些图片格式？**
A: 支持 PNG、JPG、JPEG、WebP 格式。
