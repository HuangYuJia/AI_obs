#### 2.1 初始问题
- **现象**：点击"生成"按钮后报错："无法连接到 AI 换装服务，请检查网络连接"

- **huggingface.co 无法访问**：DNS 解析超时
- **hf-mirror.com SSL 错误**：VPN 导致 HTTPS 连接不稳定
- **解决方案**：开启 VPN 后 Python 可以访问 huggingface.co

### 四、本地部署 IDM-VTON

#### 4.1 决策依据
- 用户配置满足要求：
  - GPU: GTX 1080 8GB ✓
  - 内存: 32GB ✓
  - 磁盘: 190GB 可用 ✓
  - CUDA: 12.6 ✓

#### 4.2 部署过程

**步骤 1：克隆代码**
```bash
git clone https://github.com/yisol/IDM-VTON.git
```

**步骤 2：安装依赖**
```bash
pip install accelerate==0.25.0 transformers==4.36.2 diffusers==0.25.0 einops==0.7.0 scipy==1.11.1 opencv-python gradio==4.24.0 fvcore cloudpickle omegaconf pycocotools basicsr av onnxruntime==1.16.2
```

**步骤 3：下载 checkpoints**
```
ckpt/densepose/model_final_162be9.pkl (243.9 MB)
ckpt/humanparsing/parsing_atr.onnx (254.5 MB)
ckpt/humanparsing/parsing_lip.onnx (254.5 MB)
ckpt/openpose/ckpts/body_pose_model.pth (199.6 MB)
```

**步骤 4：修复依赖问题**
```
问题：huggingface_hub 版本不兼容
解决：pip install diffusers==0.25.0 transformers==4.36.2 huggingface_hub==0.20.3 tokenizers==0.15.2
```

**步骤 5：创建本地 API 服务**
- 文件：`IDM-VTON/local_api.py`
- 端口：8444
- 功能：提供 REST API 接口

**步骤 6：下载模型权重**
```
从 Hugging Face 下载：
- UNet: 12.0 GB
- Text Encoder 1: 492 MB
- Text Encoder 2: 2.78 GB
- Image Encoder: 2.53 GB
- VAE: 335 MB
- UNet Encoder: 10.3 GB
总计：约 28 GB
```

**步骤 7：修复 torchvision 问题**
```
问题：torchvision 是 CPU 版本，不支持 CUDA
解决：pip install torchvision==0.21.0 --index-url https://download.pytorch.org/whl/cu126
```

**步骤 8：修复代码问题**
```
问题 1：utils_mask 模块导入错误
解决：sys.path.append('./gradio_demo')

问题 2：torch.cuda.amp.autocast 已弃用
解决：改为 torch.amp.autocast('cuda')

问题 3：管道调用参数格式错误
解决：参考 gradio_demo/app.py 修改参数格式
```

### 五、前端修改

#### 5.1 底部预览窗口修复
**问题**：原始直播画面不显示

**原因**：底部预览只在 AI 生成后才更新

**解决方案**：修改 `static/app.js` 中的 `renderFrame` 函数
```javascript
// 同时更新底部预览
if (!hasOverlay) {
    elements.bottomPreviewImg.src = canvas.toDataURL('image/png');
    elements.bottomPreviewImg.style.display = 'block';
    const placeholder = document.querySelector('.bottom-preview__placeholder');
    if (placeholder) placeholder.style.display = 'none';
}
```
